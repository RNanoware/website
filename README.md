# Neill Robson's Website

Source code for the personal website and blog of Neill Robson.

## Local Development

1. Download or clone the repo
  - `git clone git@github.com:rnanoware/website.git`
2. Enter the folder
  - `cd website/`
3. Start Jekyll server
  - `jekyll serve`
4. Access via
  - http://127.0.0.1:4000/
